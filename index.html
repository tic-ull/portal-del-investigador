Proceso de instalación

  1. Instalar/Actualizar el paquete VIRTUALENV

sudo apt-get install python-pip pip show virtualenv sudo pip install --upgrade virtualenv pip show virtualenv

  1. Crear y activar un VIRTUALENV para el proyecto

    mkdir -p virtualenv

    cd virtualenv

    virtualenv --no-site-packages .

    source bin/activate

  2. Descargar el repositorio

    git clone git@gitlab.stic.ull.es:investigacion/portal-de-investigacion.git

  3. Instalar dependencia (desde la raíz del proyecto y con el virtualenv activo)

    pip install -U distribute

    sudo apt-get install libmysqlclient-dev postgresql postgresql-client postgresql-contrib libpq-dev libxml2-dev libxslt1-dev python-dev

    pip install -r requirements.txt

  4. Configurar PostgreSQL

  5. Crear el esquema de la base de datos

    python manage.py makemigrations

    python manage.py migrate
